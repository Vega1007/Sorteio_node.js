Random Draw System (Node.js)

This is a simple random draw web application built with Node.js.
The system allows users to register their name and phone number to participate in a draw.

Features

Participant registration (name and phone number)

Admin page to view all registered participants

Display of total number of participants

Ability for the administrator to remove participants

Random selection of a winner

Winner page displaying the selected participant’s name and phone number

Technologies Used

Node.js

Express

SQLite

Sequelize

Express-Handlebars

Body-Parser

Express-Session

Connect-Flash

SQLite was used as the database for this project because it was my first project, created to gain experience with backend development and database integration before working with other database systems.

Running the Project

Install the dependencies:

npm install

Start the development server:

npm run dev

Open the browser and go to:

http://localhost:8000

Purpose of the Project

This project was created to practice backend development with Node.js, including database integration, server-side logic, and building a simple web application structure.
